# Sources

Every corpus chunk carries a `source_id` in its file header. This file is the citation record
those IDs map back to.

## PHASE-EVIDENCE — `01_phase_evidence.md`

- McNulty, K.L., Elliott-Sale, K.J., Dolan, E., Swinton, P.A., Ansdell, P., Goodall, S.,
